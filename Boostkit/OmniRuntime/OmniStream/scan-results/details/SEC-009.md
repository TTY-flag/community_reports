# 漏洞深度分析报告：StreamMap UDF 配置注入

## 漏洞标识

| 属性 | 值 |
|------|-----|
| **漏洞 ID** | SEC-009 |
| **严重程度** | **Critical（严重）** |
| **置信度** | 75% |
| **CWE 编号** | CWE-15（外部控制系统或组件的配置） |
| **涉及模块** | streaming_runtime_tasks, jni, core_udf（跨模块） |
| **关联漏洞** | SEC-004（同一攻击链的不同环节） |

---

## 一、漏洞概述

本漏洞是 SEC-004（UDF dlopen 代码执行）攻击链的关键环节。StreamMap 算子从任务配置中提取 UDF 库路径（`config["udf_so"]`），直接传递给 `UDFLoader::LoadMapFunction()` 进行动态库加载，全程缺乏对配置来源的验证和路径安全检查。

此漏洞与 SEC-004 形成完整的攻击链：
- **SEC-009**：配置注入点（从 JSON 配置提取恶意路径）
- **SEC-004**：代码执行点（dlopen 加载恶意库）

## 五、代码证据

### 关键代码片段

#### StreamMap::loadUdf（漏洞触发点）

```cpp
// cpp/streaming/api/operators/StreamMap.h
// Line 42-55
void loadUdf(const nlohmann::json &config)
{
    // **危险1**：直接从配置提取路径，无验证
    std::string soPath = config["udf_so"];  // Line 44
    
    // **危险2**：udf_obj 也被解析，可能包含恶意参数
    std::string udfObj = config["udf_obj"];
    nlohmann::json udfObjJson = nlohmann::json::parse(udfObj);  // Line 46
    
    std::cout << "map udf obj: " << udfObj << std::endl;
    
    // **危险3**：调用 UDFLoader，触发 dlopen
    auto *symbol = udfLoader.LoadMapFunction(soPath);  // Line 49
    if (symbol == nullptr) {
        throw std::out_of_range("null pointer when load " + soPath);
    }
    
    // **危险4**：执行 UDF 函数
    function = symbol(udfObjJson);  // Line 53
    this->userFunction = function.release();
}
```

**安全问题分析**：
1. `config["udf_so"]` 直接获取，无路径验证
2. `config["udf_obj"]` 也被解析，无异常处理
3. 无白名单检查可加载的库
4. 无签名验证机制
5. 无配置来源验证（是否来自可信的 JobManager）

#### StreamMap 构造函数（入口点）

```cpp
// cpp/streaming/api/operators/StreamMap.h
// Line 25-31
explicit StreamMap(Output *output, nlohmann::json config, bool isStream = true)
{
    LOG("-----create StreamMap-----");

    std::string soPath = config["udf_so"];

        LOG_ERROR("Configuration source not trusted");
        throw std::invalid_argument("Untrusted configuration source");
    }
    
    // ... 后续处理
}

bool isUdfPathAllowed(const std::string& path) {
    // 1. 规范化路径
    std::string canonicalPath = std::filesystem::canonical(path);
    
    // 2. 检查白名单目录
    const std::vector<std::string> allowedDirs = {
        "/opt/omnistream/udf/",
        "/usr/lib/omnistream/"
    };
    
    for (const auto& dir : allowedDirs) {
        if (canonicalPath.starts_with(dir)) {
            return true;
        }
    }
    
    return false;
}
```

#### 2. 配置字段验证

```cpp
class UdfConfigValidator {
public:
    static bool validate(const nlohmann::json& config) {
        // 1. 检查必需字段
        if (!config.contains("udf_so") || !config.contains("udf_obj")) {
            return false;
        }
        
        // 2. 检查字段类型
        if (!config["udf_so"].is_string() || !config["udf_obj"].is_string()) {
            return false;
        }
        
        // 3. 检查路径格式
        std::string soPath = config["udf_so"];
        if (soPath.find("..") != std::string::npos) {  // 路径遍历
            return false;
        }
        if (!soPath.ends_with(".so")) {  // 文件类型
            return false;
        }
        
        // 4. 检查 udf_obj 大小
        if (config["udf_obj"].get<std::string>().length() > 10240) {
            return false;
        }
        
        return true;
    }
};
```

// 对 TDD JSON 添加签名验证
class ConfigSignatureVerifier {
public:

            return false;

TEST(UdfConfigValidation, PathTraversal) {
    nlohmann::json maliciousConfig;
