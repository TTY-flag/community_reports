# SEC-008：ObjectBuffer大小未验证漏洞

## 漏洞概述

| 字段 | 值 |
|------|-----|
| **漏洞ID** | SEC-008 |
| **类型** | Improper Input Validation (CWE-20) |
| **严重性** | Critical (verified) |
| **置信度** | 85 |
| **源模块** | streaming_runtime_io |
| **发现者** | security-auditor |

## 与 VULN-STREAM-001 的关系

本漏洞与 **VULN-STREAM-001** 为同一漏洞的重复发现：
- 相同漏洞位置：`OmniAbstractStreamTaskNetworkInput.h`
- 相同漏洞类型：网络数据大小未验证导致越界访问
- 不同发现者：本漏洞由 `security-auditor` 发现，VULN-STREAM-001 由 `dataflow-scanner` 发现
- 不同分析角度：本漏洞侧重输入验证缺失，VULN-STREAM-001 侧重越界访问风险

**完整技术分析请参考 [VULN-STREAM-001.md](./VULN-STREAM-001.md)**

## 补充分析：跨模块视角

### 模块交互分析

```
数据流跨模块路径：

[模块1: runtime_buffer]
  - ObjectBuffer 类定义
  - GetSize() 返回值来自网络数据
  - ObjectSegment 类定义
  - getObject() 直接访问数组，无边界检查

       ↓ 网络数据传递

[模块2: streaming_runtime_io]
  - OmniAbstractStreamTaskNetworkInput
  - processBufferOrEventOptForSQL()
  - 使用 GetSize()/GetOffset() 进行循环迭代
  - 调用 getObject(index) 访问数据
```

### 跨模块验证缺失分析

| 模块 | 验证缺失点 | 风险 |
|------|------------|------|
| runtime_buffer | ObjectBuffer.GetSize() 来源未验证 | 返回值可信度未知 |
| runtime_buffer | ObjectSegment.getObject() 无边界检查 | 直接越界访问 |
| streaming_runtime_io | 未验证 size vs capacity | 循环超限 |
| streaming_runtime_io | 未验证 offset + size 越界 | 累加溢出 |

## 从跨模块视角的修复建议

### runtime_buffer 模块修复

```cpp
// ObjectSegment.h
StreamElement* getObject(int offset)
{
    // 添加边界检查
    if (offset < 0 || offset >= static_cast<int>(size)) {
        LOG_ERROR("ObjectSegment::getObject OOB: offset=" << offset 
                  << " capacity=" << size);
        return nullptr;
    }
    return objects_[offset];
}

// ObjectBuffer.h (具体实现类)
class NetworkObjectBuffer : public ObjectBuffer {
public:
    int GetSize() const override {
        // 修复：返回已验证的大小
        return validated_size_;
    }
    
    void SetClaimedSize(int claimed_size, int actual_capacity) {
        // 验证并设置安全大小
        validated_size_ = std::min(claimed_size, actual_capacity);
        if (claimed_size > actual_capacity) {
            LOG_WARN("Claimed size " << claimed_size 
                     << " exceeds capacity " << actual_capacity 
                     << ", capped to " << validated_size_);
        }
    }
private:
    int validated_size_;
};
```

### streaming_runtime_io 模块修复

```cpp
// OmniAbstractStreamTaskNetworkInput.h
DataInputStatus processBufferOrEventOptForSQL(...) {
    if (bufferOrEvent->isBuffer()) {
        auto buff = reinterpret_cast<ObjectBuffer*>(bufferOrEvent->getBuffer());
        auto objSegment = buff->GetObjectSegment();
        
        // 获取并验证容量
        int segment_capacity = static_cast<int>(objSegment->getSize());
        int claimed_size = buff->GetSize();
        int claimed_offset = buff->GetOffset();
        
        // 交叉验证：确保 claimed 值不超出实际容量
        if (claimed_offset >= segment_capacity) {
            LOG_ERROR("Offset " << claimed_offset << " >= capacity " << segment_capacity);
            buff->RecycleBuffer();
            return DataInputStatus::MORE_AVAILABLE;
        }
        
        int effective_size = std::min(claimed_size, segment_capacity - claimed_offset);
        
        for (int index = claimed_offset; index < claimed_offset + effective_size; index++) {
            // getObject 已添加边界检查，但仍需防御性验证
            StreamElement* object = objSegment->getObject(index);
            if (object == nullptr) {
                LOG_WARN("Null object at index " << index);
                continue;
            }
            // ... 处理 object
        }
    }
}
```

### 网络层修复建议

```cpp
// 网络数据包反序列化
void deserializeNetworkBuffer(NetworkBuffer* buffer) {
    // 解析 header
    int claimed_size = readInt(buffer);
    int claimed_offset = readInt(buffer);
    
    // 解析 payload
    int actual_segment_size = readInt(buffer);
    
    // 立即验证并拒绝非法数据包
    if (claimed_size < 0 || claimed_size > MAX_BUFFER_SIZE) {
        throw SecurityException("Invalid buffer size: " + claimed_size);
    }
    if (claimed_offset < 0 || claimed_offset > actual_segment_size) {
        throw SecurityException("Invalid offset: " + claimed_offset);
    }
    if (claimed_offset + claimed_size > actual_segment_size) {
        throw SecurityException("Size overflow: " + 
            (claimed_offset + claimed_size) + " > " + actual_segment_size);
    }

    // ...
}
```
