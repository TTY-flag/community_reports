# udf-002：环境变量注入漏洞

## 漏洞概述

**漏洞类型**: 环境变量注入 (CWE-78)  
**严重级别**: High  
**置信度**: 90%  
**影响模块**: udf

OmniOperator 在创建 JVM 时读取 `OMNI_OPERATOR_CLASSPATH` 环境变量，直接用作 JVM 选项，无任何验证或过滤。

## 漏洞触发条件

1. 攻击者能够控制目标机器的环境变量
2. 攻击者能够设置 `OMNI_OPERATOR_CLASSPATH`
3. 攻击者能够在设置的路径放置恶意代码

## 关键代码分析

### jni_util.cpp:144-149

```cpp
static void CreateJavaVM()
{
    ...
    JavaVMOption options[1];
    // ⚠️ 直接读取环境变量，无验证
    auto classPath = getenv("OMNI_OPERATOR_CLASSPATH");
    if (classPath == nullptr) {
        LogError("Create JVM failed since OMNI_OPERATOR_CLASSPATH is not set");
        return;
    }
    // ⚠️ 直接用作 JVM 选项
    options[0].optionString = classPath;
    ...
}
```

## 利用分析

### JVM 选项注入可能性

根据项目模型，此环境变量标记为 `trusted_admin`，由管理员设置。但如果攻击者能控制环境变量：

```bash
# 攻击者设置环境变量
export OMNI_OPERATOR_CLASSPATH="-Djava.security.manager=none /path/to/malicious.jar"
# 或注入其他 JVM 选项
export OMNI_OPERATOR_CLASSPATH="-Djava.rmi.server.hostname=attacker.com ..."
```

### 攻击场景

| 攻击者能力 | 攻击方式 |
|-----------|---------|
| SSH 访问 | 直接设置环境变量 |
| 脚本注入 | 在启动脚本中添加 export |
| 容器环境 | 控制 Docker ENV |
| CI/CD 访问 | 在部署配置中注入 |

## 危害评估

### 攻击影响
- **JVM 选项注入**: 可禁用安全管理器或注入恶意配置
- **恶意类路径**: 可将恶意 jar 加入类路径
- **权限提升**: 通过 JVM 选项获取更高权限

### CVSS 评分预估

由于环境变量需要管理员级别控制，攻击难度较高：
- **攻击向量**: Local (需要机器访问)
- **攻击复杂度**: Medium
- **权限要求**: High (需要设置环境变量的权限)
- **影响**: High

**预估 CVSS 评分**: 5.0 (Medium)

## 修复建议

### 1. 环境变量验证（优先级：中）

```cpp
static void CreateJavaVM()
{
    auto classPath = getenv("OMNI_OPERATOR_CLASSPATH");
    if (classPath == nullptr) {
        LogError("...");
        return;
    }
    
    // 验证格式
    if (!IsValidClassPath(classPath)) {
        LogError("Invalid OMNI_OPERATOR_CLASSPATH format");
        return;
    }
    
    // 验证路径存在
    if (!PathExists(classPath)) {
        LogError("Classpath path does not exist");
        return;
    }
    
    // 使用验证后的值
    options[0].optionString = classPath;
}
```

### 2. 使用安全配置文件（优先级：高）

```cpp
// 从安全配置文件读取 classpath，而非环境变量
auto classPath = ReadFromSecureConfig("classpath");
```

### 3. 限制 JVM 选项（优先级：中）

```cpp
// 使用固定的 JVM 选项模板，只允许特定格式
std::string safeClassPath = "-Djava.class.path=";
if (classPath != nullptr && IsValidPath(classPath)) {
    safeClassPath += classPath;
}
options[0].optionString = safeClassPath.c_str();
```
