# DF-012: TransformerModel 模型加载数据流

## 漏洞概述

**漏洞 ID**: DF-012  
**类型**: Code Injection (CWE-94)  
**严重程度**: Critical  
**置信度**: 80 (CONFIRMED)  
**主报告关联**: SEC-011 (模型实例化数据流)

---

## 漏洞定位

**文件**: `tensor_cast/transformers/model.py`  
**行号**: 185-192  
**函数**: `TransformerModel.__init__`

---

## 数据流追踪

### 核心代码路径

```python
# TransformerModel.__init__ (185-194)
self._inner = auto_loader.load_model(
    self.hf_config,
    self.model_config.dtype,
    trust_remote_code=self.model_config.trust_remote_code,  # ← 关键参数
)

# 或自动加载路径 (191-194)
self.hf_config, self._inner = auto_loader.auto_load_model_and_config(
    self.model_id, self.model_config
)
```

---

## 两条执行路径

**路径 A: 预配置加载**
```
model_config.trust_remote_code (上游设置)
    → load_model(trust_remote_code=True)
    → try_to_load_model
    → AutoModel.from_config
    → RCE
```

**路径 B: 自动加载**
```
model_config (含 trust_remote_code 默认值)
    → auto_load_model_and_config
    → load_config + load_model
    → 两路径都可能触发 RCE
```

---

## 数据来源分析

| 数据字段 | 来源 | 传递路径 |
|----------|------|----------|
| `model_config.trust_remote_code` | ModelConfig 默认值 | SEC-011 |
| `model_config.trust_remote_code` | ConfigResolver 设置 | SEC-012 |
| `model_config.trust_remote_code` | 用户显式设置 | CLI 参数 |

无论哪种来源，TransformerModel 都会无条件传递。
