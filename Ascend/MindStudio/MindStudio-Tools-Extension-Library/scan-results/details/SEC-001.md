## 核心漏洞代码

```c
// c/include/mstx/mstx_detail/mstx_impl.h:178-181

char *injectionPath = getenv(MSTX_INJECTION_PATH);  // 直接读取环境变量
if (injectionPath) {
    handle = dlopen(injectionPath, RTLD_LAZY | RTLD_LOCAL);  // 无验证直接加载
}
```
