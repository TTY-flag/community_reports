# Arbitrary Library Load Vulnerability: Untrusted .so File Loading via importlib

## Vulnerability Overview

| Property | Value |
|----------|-------|
| **Vulnerability ID** | 4800ec024e83 |
| **Vulnerability Type** | Arbitrary Library Load (CWE-114: Injection of Arbitrary Code via Library Loading) |
| **Severity** | High |
| **Confidence** | 85% → **Confirmed as Independent Vulnerability** |
| **Affected Versions** | MindStudio-Kernel-Launcher (current version) |
| **Attack Complexity** | Low |
| **Impact Scope** | Arbitrary Code Execution with Python Process Privileges |

### Vulnerability Summary

MindStudio-Kernel-Launcher's `compile()` function with `use_cache=True` bypasses the build script execution and directly loads an arbitrary `.so` file via `importlib.util.spec_from_file_location()`. The loaded library's initialization code executes with Python process privileges. While `FileChecker` validates file metadata, it does NOT verify library content or integrity, allowing attackers to load pre-placed malicious shared libraries.

## FileChecker Limitations Analysis

## Remediation Recommendations

### 1. Library Loading Validation (Priority: HIGH)

Implement library content/integrity verification before loading:

```python
import hashlib
import struct
import os

def validate_shared_library(library_path: str) -> bool:
    """
    Validate shared library before loading via importlib.
    Performs structural and integrity checks.
    """
    
    # 1. Path validation - restrict to allowed directories
    ALLOWED_DIRS = [
        os.path.expanduser("~/.mskl/kernels"),
        "/usr/local/lib/mskl",
        "/opt/mskl/lib",
    ]
    
    real_path = os.path.realpath(library_path)
    allowed = any(real_path.startswith(d) for d in ALLOWED_DIRS)
    if not allowed:
        raise SecurityError(f"Library path not in allowed directories: {library_path}")
    
    # 2. ELF structure validation
    with open(real_path, 'rb') as f:
        magic = f.read(4)
        if magic != b'\x7fELF':
            raise SecurityError(f"Invalid ELF magic: {library_path}")
        
        # Check ELF header for anomalies
        elf_class = f.read(1)  # 1=32-bit, 2=64-bit
        if elf_class not in [b'\x01', b'\x02']:
            raise SecurityError(f"Invalid ELF class: {library_path}")
    
    # 3. Owner validation (block, not warn)
    file_stat = os.stat(real_path)
    if os.getuid() != 0 and os.getuid() != file_stat.st_uid:
        raise SecurityError(f"Library not owned by current user: {library_path}")
    
    # 4. No soft links allowed (block, not warn)
    if os.path.islink(library_path) or os.path.islink(real_path):
        raise SecurityError(f"Soft links not allowed: {library_path}")
    
    # 5. Digital signature verification (if available)
    # ... implement signature check if keys are available
    
    return True

# Apply in compile() before CompiledKernel creation
def compile(...):
```

### 2. Allowed Directory Whitelist (Priority: HIGH)

Restrict library loading to specific directories:

```python
class KernelLibraryManager:
    """
    Manage allowed kernel library directories.
    Libraries can only be loaded from whitelisted paths.
    """
    
    ALLOWED_BASE_DIRS = [
        '/tmp/mskl_workspace',  # User workspace (with restrictions)
        os.path.expanduser('~/.mskl/cache'),  # User cache
    ]
    
    @classmethod
    def is_path_allowed(cls, library_path: str) -> bool:
        """Check if library path is in allowed directories."""
        real_path = os.path.realpath(library_path)
        
        # Must be in allowed directory
        for allowed_dir in cls.ALLOWED_BASE_DIRS:
            if real_path.startswith(os.path.realpath(allowed_dir)):
                return True
        
        return False
    
    @classmethod
    def create_safe_library_dir(cls) -> str:
        """Create a secure directory for kernel libraries."""
        safe_dir = os.path.expanduser('~/.mskl/cache')
        os.makedirs(safe_dir, mode=0o700, exist_ok=True)
        return safe_dir
```
